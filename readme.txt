== Description ==

Email Custom Fields with predesignated templates. Different template for different email.

FEATURES

* Select template from list and run it with shortcode
* Templates will send email. You can add your css to style it
* No additional setup for Email Custom Fields. Everything is preset in template
* ReCAPTCHA for [contact01] mail form
* Newsletter form for uses ajax and it doesn't reload page

==Installation==


AUTOMATIC INSTALLATION
1. Search for Email Custom Fields through 'Plugins > Add New' interface.
2. Find the plugin box of Email Custom Fields and click on the 'Install Now' button.
3. Then activate the Email Custom Fields plugin.
4. Copy the shortcode echo do_shortcode('[newsletter]')   and paste it into a theme file.


MANUAL INSTALLATION
1. Download Email Custom Fields
2. Upload the Email Custom Fields through 'Plugins > Add New > Upload' interface or upload frontpage-slider folder to the '/wp-content/plugins/' directory.
3. Activate the Email Custom Fields plugin through the 'Plugins' menu in WordPress.
